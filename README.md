## Présentation

Diakhou Hair & Beauty est une plateforme e-commerce premium conçue pour offrir une expérience d'achat luxueuse, fluide et performante. Ce projet utilise React, TypeScript, Tailwind CSS et Vite pour une interface utilisateur réactive et des animations sophistiquées.

## Caractéristiques

- **Design Premium & Glassmorphism** : Composants modernes, luxueux et épurés
- **Animations Avancées** : Transitions fluides 
- **Catalogue Dynamique** : Filtrage et recherche avancée de produits
- **Responsive** : Parfaitement adapté pour mobile, tablette et ordinateur de bureau

## Installation et Lancement Local

**Prérequis :** Node.js (version 18+)

1. Cloner le projet et installer les dépendances :
   ```bash
   npm install
   ```

2. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```
   L'application sera accessible sur `http://localhost:5173` ou `http://localhost:3001` (selon la configuration).

3. Construire le projet pour la production :
   ```bash
   npm run build
   ```

## Technologies Utilisées

- **Frontend Framework**: React 18 (via Vite)
- **Langage**: TypeScript
- **Styling**: Tailwind CSS
- **Routage**: React Router DOM
- **Icônes**: Lucide React
